# Sales & Revenue Analysis – SQL + Excel + Power BI

### Tools: SQL | Excel | Power BI

## Overview
This project analyzes sales performance over time to identify trends, top-performing products, and regional performance.

## Key Tasks
- Cleaned and prepared sales data
- Wrote SQL queries to calculate revenue, profit, and trends
- Used Excel for exploratory analysis and validation
- Built a Power BI dashboard to visualize KPIs

## Files
- `data_raw/` – raw input data
- `data_clean/` – cleaned dataset
- `SQL_queries.sql` – main analysis queries
- `Sales_Report.xlsx` – Excel analysis
- Power BI file – dashboard (to be added)

## Insights (to be completed)
- Top regions/products by revenue
- Revenue trend over months
- Key performance drivers
